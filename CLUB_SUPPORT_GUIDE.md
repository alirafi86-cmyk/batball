## Quick Start

### I Just Want to Score a Match
1. Ask the **match admin** for the **Scorer PIN**
2. Open https://batball-three.vercel.app/ on your phone
3. Click **"Resume Match"**
4. Enter the Scorer PIN
5. Select striker, bowler, runs, and wickets for each ball
6. Click **"Finish Match"** when done

### I Want to View a Match from WhatsApp
1. Download the JSON file from WhatsApp
2. Open https://batball-three.vercel.app/
3. Click **"Open Shared Scorecard"**
4. Select the downloaded file
5. View full stats and match summary
