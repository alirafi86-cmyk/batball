## Live Scoring (1 hour)

### The UI
- **Top**: Current score, overs, batter stats
- **Middle**: Recent ball results (last 6 balls)
- **Bottom**: Run buttons (0,1,2,3,4,6) and extras

### How to Score
1. **Select Batter**: Click if needed (auto-selected first 2)
2. **Select Bowler**: Click to change
3. **Score a run**: Click run buttons (0, 1, 2, 3, 4, 6)
4. **Wicket**: Select wicket type from modal
5. **Extras**: Toggle wide/no-ball before clicking runs

### Tips
- **Auto-save**: Score saved every second to localStorage
- **Offline**: Works completely without internet
- **Recovery**: Close accidentally â†’ Click "Resume" to continue
- **Undo**: Click "Undo" button to reverse last ball

## Troubleshooting

### Match Won't Start?
- Check internet connection briefly (for first setup)
- Verify all 11 players selected per team
- Try refreshing page

### Stats Not Showing?
- Match must be saved to "Club Hub"
- Refresh page to recalculate
- Stats appear instantly after save

### Squads Disappeared?
- Browser cache cleared? Restore from backup
- Try different browser
- Backup regularly via Club Hub â†’ Storage

### Image Export Failing?
- Ensure match was saved with data
- Try again in different browser
- Close other apps to free memory
